Project 1: Hand Exoskeleton for Rehabilitation

Vision-Based Control for Mirror Therapy and Assisted Movement

Figure 1: Conceptual design of the underactuated hand exoskeleton.

Description & Objectives

This project involved the design, modeling, and control of a lightweight, underactuated hand exoskeleton intended for post-stroke rehabilitation. The primary objective was to create a system capable of providing assisted finger flexion/extension based on the mirroring of a healthy hand's pose, facilitating motor relearning through mirror therapy paradigms.

Modeling and Control Methods

  • Mechanical Design: Developed using CAD software, featuring a tendon-driven mechanism for underactuation, reducing weight and control complexity.
  • Dynamic Modeling: Established a kinematic model of the finger linkages and tendon paths to relate actuator displacement to joint angles.
  • Control Strategy: Implemented a proportional-derivative (PD) controller for position control of the actuator, with gains tuned for smooth and stable motion.

Vision-Based Hand Pose Mirroring

A computer vision pipeline was developed using OpenCV and MediaPipe. The system captures real-time video of the user's healthy hand, extracts 21 3D landmark positions for each finger, and maps these coordinates to desired joint angles for the exoskeleton. This creates a direct biomimetic control loop.

Quantitative Results & Performance

Metric Result Implication
Mean Joint Angle Tracking Error 4.2° ± 1.8° High fidelity in pose replication.
End-to-End System Latency 120 ms Sufficient for quasi-real-time therapeutic interaction.
Range of Motion (ROM) Assisted 0-85° per MCP joint Covers functional grasping ROM.
Force Exertion at Fingertip Up to 8 N Adequate for assisted movement against mild spasticity.

Conclusion: The project successfully demonstrated a proof-of-concept for a vision-driven, low-cost rehabilitation exoskeleton. Results validate the feasibility of using consumer-grade cameras for intuitive control of assistive devices, paving the way for more accessible home-based therapy solutions.

Project 2: Fuzzy Logic Based Control of a Unicycle Mobile Robot

Navigation and Path Following in Unstructured Environments

Figure 2: Simulation snapshot of the fuzzy-controlled robot tracking a reference path.

Control Strategies & Comparative Analysis

This research focused on designing and comparing control architectures for a non-holonomic unicycle robot tasked with path following and obstacle avoidance. The core comparison was between a classical PID controller and a Mamdani-type Fuzzy Logic Controller (FLC).

  • Classical PID Control: Designed based on a linearized kinematic model. While effective for simple trajectories, performance degraded significantly with sharp turns and in the presence of simulated sensor noise.
  • Fuzzy Logic Controller: Designed with two input linguistic variables (tracking error and heading error) and two outputs (linear and angular velocity). Membership functions and a rule base of 25 rules were developed to encapsulate expert navigation knowledge.

Performance Results & Discussion

The systems were evaluated in a MATLAB/Simulink simulation environment across multiple scenarios: straight-line following, sinusoidal paths, and navigation with static obstacles.

Scenario PID Controller (Avg. Error) Fuzzy Controller (Avg. Error) Improvement
Straight Line 0.05 m 0.03 m 40%
Sinusoidal Path 0.18 m 0.07 m 61%
Obstacle Avoidance Collision (in 2/5 trials) No collisions Superior robustness

The Fuzzy Logic Controller exhibited markedly superior performance in handling non-linearity and uncertainty. It achieved smoother control actions, reduced steady-state error in complex trajectories, and demonstrated inherent robustness to perturbations, validating the advantage of rule-based, intelligent control for autonomous mobile systems in dynamic settings.
