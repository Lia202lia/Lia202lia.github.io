Project 1: Vision-Based Hand Exoskeleton for Rehabilitation

Rehabilitation Robotics Computer Vision Control Systems 2023-Present

Figure 1: System architecture showing vision capture, processing, and actuation components

Problem Statement

Post-stroke rehabilitation requires repetitive, guided movement therapy. Traditional methods are labor-intensive and expensive, limiting accessibility. Existing robotic systems lack intuitive control and affordability for home use.

Technical Approach

1

Mechanical Design

Underactuated tendon-driven mechanism reducing weight by 40% compared to fully actuated designs. 3D-printed components for rapid prototyping.

2

Vision System

MediaPipe hand tracking with custom calibration for individual anatomical variations. Real-time 3D pose estimation at 30fps.

3

Control Algorithm

PD controller with gravity compensation. Adaptive gain scheduling based on user capability assessment.

4

System Integration

ROS-based architecture connecting perception, planning, and control modules. Modular design for component testing.

Quantitative Results

Performance Metric Result Methodology Significance
Joint Angle Tracking Error 4.2° ± 1.8° Comparison with goniometer measurements Clinically acceptable for therapy
System Latency 120ms end-to-end High-speed camera timestamp comparison Enables real-time mirror therapy
Range of Motion 0-85° MCP joints Physical measurement with protractor Covers functional grasping range
Force Generation 8N at fingertip Load cell measurement Sufficient for assisted movement

Impact & Applications

  • Clinical: Enables home-based rehabilitation with professional guidance quality
  • Economic: Reduces therapy costs by estimated 40% through automation
  • Technical: Demonstrates effective integration of vision and actuation in soft robotics
  • Research: Provides framework for adaptable rehabilitation systems

Conclusion

This project successfully demonstrates a vision-controlled rehabilitation system with clinically relevant performance metrics. The integration of consumer-grade vision with custom mechanical design presents a viable path toward accessible robotic therapy.

Project 2: Fuzzy Logic Control for Mobile Robot Navigation

Intelligent Control Mobile Robotics Simulation 2022

Figure 2: Path following performance comparison: PID vs Fuzzy Logic Controller

Research Question

Can fuzzy logic control provide superior performance to classical PID for non-holonomic mobile robots in unstructured environments with uncertainty?

Methodology

Controller Design

PID Controller
  • Based on linearized kinematic model
  • Fine-tuned for straight-line tracking
  • Fixed gains throughout operation
Fuzzy Logic Controller
  • Mamdani-type inference system
  • 25 rules based on expert knowledge
  • Adaptive to environmental changes

Simulation Environment

  • MATLAB/Simulink implementation
  • Three test scenarios: straight line, sinusoidal path, obstacle field
  • Added sensor noise (5% Gaussian) to simulate real conditions
  • Performance metrics: tracking error, completion time, collision rate

Performance Analysis

Straight Line Error
0.18m
0.05m
Sinusoidal Path Error
0.28m
0.11m
Obstacle Success Rate
60%
100%

Key Findings

  1. Fuzzy controller showed 61% lower average error in complex trajectories
  2. Superior handling of non-linearities and uncertainty
  3. No collisions in obstacle field vs 40% collision rate for PID
  4. Smoother control actions with less overshoot

Research Contribution

This study validates fuzzy logic as a robust control strategy for mobile robots in dynamic environments. The quantitative comparison provides evidence for intelligent control methods over traditional approaches when dealing with uncertainty and non-linearity.
