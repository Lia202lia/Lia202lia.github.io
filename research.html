Rehabilitation Robotics

Adaptive Hand Exoskeleton for Post-Stroke Rehabilitation

Role: Lead Designer & Controls Engineer
Timeline: 8 months
Tools: SolidWorks, MATLAB, Arduino, EMG Sensors

Overview

Designed and prototyped a lightweight, underactuated exoskeleton to assist hand opening and closing for post-stroke patients. The system uses surface EMG signals to detect user intent and provides proportional assistance.

Problem Statement

Traditional rehabilitation requires intensive therapist supervision. Existing robotic devices are often costly, rigid, or lack adaptive support for varying patient capabilities.

Technical Approach

  • Underactuated tendon-driven mechanism (3 actuators for 5 fingers)
  • EMG signal processing for intent classification
  • Proportional-derivative control with force feedback
  • 3D-printed compliant components for safety

Quantitative Results

92%
Intent detection accuracy
380g
Total system weight
30%
Reduction in perceived exertion
25N
Max assistive force per finger

Impact & Contribution

This project demonstrates a cost-effective approach to adaptive rehabilitation robotics. The underactuated design reduces complexity while maintaining functionality, making such technology more accessible in resource-limited settings.

Intelligent Control

Fuzzy Logic Control for Non-Holonomic Mobile Robot Navigation

Role: Control Systems Designer
Timeline: 4 months
Tools: MATLAB/Simulink, ROS, Python

Overview

Developed and validated a two-layer fuzzy logic controller for precise trajectory tracking of a unicycle mobile robot, comparing performance against conventional PID control under various disturbance conditions.

Challenge

Non-holonomic constraints and system uncertainties make precise control challenging. Traditional controllers often struggle with nonlinearities and disturbances.

Methodology

  • Mamdani-type fuzzy inference system
  • Two-layer architecture: outer (pose error to velocities) and inner (velocity tracking)
  • Centroid defuzzification method
  • Comparative analysis with PID under identical conditions

Performance Metrics

68%
Error reduction vs PID
±2cm
Steady-state accuracy
20%
Mass disturbance tolerance
120ms
Worst-case computation time

Academic Contribution

This work provides quantitative evidence for fuzzy logic superiority in handling nonlinear robotic systems with uncertainties. The modular controller design can be adapted to various mobile platforms.

Computer Vision

Vision-Based Autonomous Navigation in Dynamic Indoor Environments

Role: Perception & Planning Developer
Timeline: 6 months
Tools: ROS, OpenCV, Python, Gazebo

Overview

Implemented a complete autonomous navigation stack using only an RGB-D camera. The system performs SLAM, dynamic obstacle detection, and real-time path planning without external localization aids.

Technical Stack

  • ORB-SLAM2 for localization and mapping
  • YOLOv4 for real-time object detection
  • A* algorithm for global path planning
  • Dynamic Window Approach (DWA) for local obstacle avoidance

Innovation

Integrated semantic understanding (object classification) with geometric navigation, allowing the robot to distinguish between static furniture and dynamic obstacles like people.

System Performance

15 FPS
Real-time processing
±5cm
Mapping accuracy (10m)
99%
Obstacle avoidance success
120ms
Replanning latency

Research Significance

Demonstrates a cost-effective, sensor-minimal approach to autonomous navigation suitable for service robots in human environments. The integration of semantic perception enhances safety and predictability.

Bio-inspired Robotics

Flapping-Wing Micro Air Vehicle: Design & Control

Role: Mechatronics Lead
Timeline: 7 months
Tools: SolidWorks, ANSYS, C++, 3D Printing

Overview

Designed, fabricated, and controlled a bio-inspired flapping-wing micro air vehicle (FWMAV) capable of stable hovering and basic maneuvers, mimicking insect flight mechanics.

Design Challenges

  • Ultra-lightweight structure (<30g)
  • Efficient power transmission for flapping
  • Passive stability in turbulent conditions
  • Minimal sensor suite for attitude control

Technical Solutions

  • Carbon fiber frame with mylar wings
  • Four-bar linkage for wing motion conversion
  • Cascaded PID controller using IMU data
  • Custom PCB integrating motor drivers and sensors

Flight Performance

Broader Applications

FWMAVs have potential for indoor inspection, search-and-rescue in confined spaces, and environmental monitoring. This project advances understanding of lightweight actuation and stabilization for bio-inspired robotics.
