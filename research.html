Research & projects

Each project below is presented as a concise case study, emphasizing the problem, methods, and quantitative results.

Hand Exoskeleton for Rehabilitation

Assistive exoskeleton for post-stroke upper-limb rehabilitation with adaptive, patient-specific control strategies.

Problem

Conventional hand rehabilitation often requires intensive therapist involvement and cannot easily provide consistent, high-repetition, task-specific training tailored to each patient’s evolving capabilities.

Methods & approach

  • Designed a lightweight, tendon-driven exoskeleton for the hand with mechanically decoupled fingers to allow natural grasp patterns.
  • Implemented embedded control on a microcontroller platform, integrating position sensors and current sensing for torque estimation.
  • Developed an assist-as-needed control scheme that adjusts assistance level based on measured joint trajectories and interaction forces.

Results

  • Achieved repeatable finger flexion–extension trajectories with tracking errors under 5% of the commanded range of motion across sessions.
  • Reduced peak interaction torque by approximately 20% when using adaptive assistance compared with fixed-gain control in bench-top trials.
  • Demonstrated stable operation over multiple 30-minute training sessions without actuator overheating or loss of calibration.

Fuzzy Logic Control of a Unicycle Mobile Robot

Intelligent control design for a nonholonomic mobile robot navigating toward goals under uncertainty.

Problem

Classical controllers for unicycle-type robots can struggle with model uncertainties, external disturbances, and the need for smooth, human-friendly trajectories in cluttered environments.

Methods & approach

  • Modeled the robot as a nonholonomic unicycle and defined tracking errors in polar coordinates relative to a target pose.
  • Designed a fuzzy logic controller with linguistic rules that map distance and heading errors to linear and angular velocity commands.
  • Compared the fuzzy controller against a conventional proportional controller in simulation across multiple goal configurations.

Results

  • Reduced overshoot in heading angle by more than 30% while maintaining comparable convergence times to the target pose.
  • Generated smoother trajectories with fewer abrupt velocity changes, improving suitability for human–robot shared spaces.
  • Maintained stable convergence under perturbations in wheel radius and baseline parameters of up to 15% from nominal values.

Vision-Based Gesture Recognition for Human–Robot Interaction

Real-time recognition of hand gestures to enable intuitive, contactless control of robotic systems.

Problem

Many assistive robots rely on physical interfaces that can be difficult or tiring to use for individuals with limited mobility, making it important to explore accessible, vision-based alternatives.

Methods & approach

  • Developed a camera-based pipeline for hand detection, segmentation, and gesture classification using conventional computer vision and lightweight learning models.
  • Implemented temporal smoothing to reduce spurious gesture detections and stabilize robot responses.
  • Integrated the system with a simple robot control interface to map a small vocabulary of gestures to discrete commands.

Results

  • Achieved gesture classification accuracy above 93% on a held-out test set collected under varying lighting conditions.
  • Maintained end-to-end latency under 120 ms on modest processing hardware, enabling responsive interaction.
  • Observed a reduction in unintended command activations compared with a baseline thresholding approach during pilot trials.

Intelligent Control for a Two-Link Robotic Arm

Adaptive control of a planar manipulator for accurate trajectory tracking under payload variations.

Problem

Industrial and assistive manipulators often face changing payloads and unmodeled dynamics, which can degrade tracking performance if the controller is not robust or adaptive.

Methods & approach

  • Formulated the dynamics of a two-link robotic arm using the Euler–Lagrange method and implemented a simulation environment.
  • Designed an adaptive control scheme that estimates uncertain parameters online and updates control gains accordingly.
  • Evaluated the controller on point-to-point and continuous trajectories with varying payload masses at the end effector.

Results

  • Reduced steady-state position error to less than 1 degree at each joint under payload changes of up to 25%.
  • Maintained smooth torque profiles without excessive oscillation, supporting safe physical interaction scenarios.
  • Demonstrated faster convergence to desired trajectories compared with fixed-gain PD control in simulation studies.

Embedded Control Platform for Assistive Robotics

Real-time embedded architecture for integrating sensors and actuators in compact assistive devices.

Problem

Assistive devices require reliable, low-latency control on hardware with limited computational and power resources, while supporting flexible integration of sensors and actuators.

Methods & approach

  • Designed a modular embedded architecture around a microcontroller platform with support for motor drivers, encoders, and force sensors.
  • Implemented a state-machine-based firmware structure with prioritized interrupt handling for critical control loops.
  • Developed a serial communication interface for logging data and tuning control parameters during experiments.

Results

  • Achieved stable control loop execution at 1 kHz with jitter below 5% in timing analysis.
  • Reduced wiring complexity and integration time when adding new sensors or actuators in subsequent prototypes.
  • Enabled detailed logging of system states, supporting more systematic debugging and controller tuning.

Simulation Environment for Human-in-the-Loop Robotics

Integrated simulation tools to prototype and evaluate human–robot interaction concepts before hardware deployment.

Problem

Early-stage human–robot interaction concepts are difficult to test safely and efficiently on hardware alone, especially when resources are limited.

Methods & approach

  • Set up a simulation environment combining robot dynamics, basic environment models, and simplified representations of human input.
  • Implemented scenarios for shared-control navigation and assistive manipulation tasks.
  • Defined quantitative metrics for tracking performance, user effort proxies, and safety-related measures such as minimum distance to obstacles.

Results

  • Enabled rapid evaluation of alternative control policies and interface designs before hardware implementation.
  • Identified parameter ranges that balanced responsiveness and stability, guiding later controller deployment.
  • Provided a platform for future user studies that can combine simulated and hardware components in a controlled manner.
