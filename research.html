Honors Thesis Project

Intelligent Hand Exoskeleton for Rehabilitation

Design, dynamic modeling, and control of a cable-driven hand exoskeleton with stereo vision-based hand pose mirroring

January 2024 – August 2025 Manara University Thesis Score: 96/100

Project Objective

Develop an affordable, lightweight hand exoskeleton for post-stroke rehabilitation that provides safe human-robot interaction while achieving precise trajectory tracking through intelligent control strategies.

Key Achievements

  • <0.5° tracking error in joint angle control
  • 20% reduction in settling time under disturbances
  • 15.4mm mean joint position error (from 22.4mm baseline)
  • <200ms total system latency

System Design & Modeling

Mechanical Design

  • Self-alignment mechanism for natural finger movement
  • Lightweight carbon fiber structure (total weight: 280g)
  • Cable-driven actuation with Bowden cable transmission
  • Custom 3D-printed finger linkages

Dynamic Modeling

  • Lagrangian formulation for multi-body dynamics
  • Nonlinear friction compensation in cable transmission
  • Human hand biomechanics integration
  • MATLAB/Simulink simulation environment

Control Strategies

Adaptive Sliding Mode Control

Primary

Robust control for nonlinear system with parameter uncertainties

Tracking Error: 0.4° Settling Time: 1.2s

Technical Implementation

MATLAB - Adaptive Sliding Mode Controller 

% Adaptive Sliding Mode Control for Hand Exoskeleton
function u = adaptiveSMC(theta_d, theta, dtheta, params)
    % Sliding surface definition
    s = dtheta - dtheta_d + lambda*(theta - theta_d);
    
    % Adaptive gain estimation
    phi_hat = updateAdaptiveLaw(s, params);
    
    % Control law
    u = -K*s - phi_hat'*Y(theta, dtheta) - eta*sign(s);
    
    % Saturation for safety
    u = saturate(u, u_max, u_min);
end
Python - Hand Pose Estimation 

import torch
import torch.nn as nn

class HandPoseNet(nn.Module):
    def __init__(self):
        super(HandPoseNet, self).__init__()
        self.conv1 = nn.Conv2d(3, 64, kernel_size=3, padding=1)
        self.conv2 = nn.Conv2d(64, 128, kernel_size=3, padding=1)
        self.fc1 = nn.Linear(128*28*28, 512)
        self.fc2 = nn.Linear(512, 21*3)  # 21 joints * 3 coordinates
        
    def forward(self, x):
        x = torch.relu(self.conv1(x))
        x = torch.max_pool2d(x, 2)
        x = torch.relu(self.conv2(x))
        x = x.view(x.size(0), -1)
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x.view(-1, 21, 3)
